高機能 関数電卓アプリ
ブラウザを開くだけですぐに使える、シンプルかつパワフルな関数電卓アプリです。日常的なお買い物や割り勘の計算から、専門的な数学の計算まで、これ一つでスムーズに行えます。

主な特長
豊富な関数・定数機能
三角関数（sin, cos, tan）や対数（log, ln）、さらに円周率（π）やネイピア数（e）などの定数入力に対応。理数系の学習や実務をしっかりサポートします。

便利なメモリー機能搭載
計算結果を一時的に記憶・呼び出しできる機能（M+, M-, MR, MC）を備えています。複数の計算結果を合算するような複雑な処理も簡単に行えます。

高度な計算もワンタッチ
累乗（x²）、平方根（√）、階乗（n!）、パーセント（%）など、よく使う特殊な計算もボタン一つで瞬時に答えを出します。

直感的で使いやすいデザイン
PCでもスマートフォンでも軽快に動作します。数字、四則演算、メモリー、特殊関数など、役割ごとにボタンを色分けし、どこに何があるか直感的に分かるレイアウトを採用しました。

柔軟な入力サポート
入力間違いを1文字だけ消せる「DEL」ボタン、正負を反転させる「±」ボタン、計算の優先順位をコントロールできるカッコ「( )」ボタンなど、かゆいところに手が届く機能を網羅しています。

便利な使い方・Tips
特殊ボタンの挙動:
画面に数字（または数式）を入力した状態で「x²」や「√」「sin」などの特殊ボタンを押すと、その時点の計算結果に対して即座に処理が実行されます。
例：「5 + 5」と入力して「x²」を押すと、10の2乗である「100」が表示されます。

三角関数の計算:
三角関数（sin, cos, tan）は「度（°）」での計算に対応しています。
例：「90」と入力して「sin」を押すと「1」になります。
